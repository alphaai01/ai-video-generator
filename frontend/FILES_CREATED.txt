SOURCE CODE - HOOKS
═══════════════════════════════════════════════════════════════════════════════

src/hooks/useVideoGeneration.ts
   - Custom hook managing video generation lifecycle
   - State: prompt, image, status, progress, videoUrl, jobId, error, settings
   - Methods:
     * generateFromText(prompt: string)
     * generateFromImage(file: File, prompt: string)
     * resetGeneration()
   - Auto-polling when job is submitted
   - Handles both text and image-based generation

src/hooks/useVoiceInput.ts
   - Custom hook for voice recording and transcription
   - State: isRecording, recordingDuration, transcribedText, isTranscribing, error
   - Methods:
     * startRecording()
     * stopRecording() - returns Promise<string | null>
     * clearTranscription()
   - Uses browser MediaRecorder API
   - Automatic speech-to-text conversion on stop

HOW TO USE
═══════════════════════════════════════════════════════════════════════════════

1. INSTALL DEPENDENCIES
   npm install

2. CONFIGURE (Optional)
   cp .env.example .env.local
   (Default already points to localhost:4000)

3. START DEVELOPMENT SERVER
   npm run dev

4. OPEN IN BROWSER
   http://localhost:3000

5. USE THE APPLICATION
   - Choose input method (Text/Image/Voice)
   - Enter prompt or upload content
   - Adjust settings
   - Click "Generate Video"
   - Watch progress in real-time
   - Download or view history
